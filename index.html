FOUNDER

「おもしろき こともなき世を おもしろく」を体現するリスクテイカー！

代表理事 安形 真 (Makoto Agata)

内閣府地域活性化伝道師 / (公財)えひめ西条つながり基金事務局長 / ㈱Democracy代表取締役 / 東海若手起業塾理事

愛知県出身。会計事務所を退職後、農業で起業。D2Cによる販売や農家レストランなど当時の先進的な農家のビジネスを展開。その後体調不良を機に支援業にピボット。

縁もゆかりもなかった愛媛県西条市に飛び込み、地域に根を張る「挑戦の支援者」としての歩みをスタート。社名の由来でもあるラテン語「Risicare（勇気をもって試みる）」を地で行くスタイルで、地方の停滞した空気をハックし続けている。

2019年にリズカーレを設立後、西条市のローカルベンチャー誘致・育成事業を牽引。コワーキングスペース「サカエマチHOLIC」の運営を通じて、延べ300件以上の起業支援を行い、3000万円以上の資金調達を支援し、100名を超える熱狂的なコミュニティを構築した。

CORE VALUES

商売人としての“当たり前”を、誰よりも熱く。

地域で商売をするということ。それは、いかにリピートしてもらうかということ。つまり、揺るぎない「信頼関係」を結ぶってことだ。当たり前の価値を、当たり前に出し続ける。

表面的なSNSのフォロワー数？バズ狙いのコンテンツ？
いや、私たちが求めるのはそこじゃない。社会を変えるという確かな実感を伴う、直接的な繋がりだ。

どんなに小さなチャレンジでもいい。その小さな成功体験がもたらす「自信」が、数年後のとてつもない大きな変化に繋がると、私たちは本気で信じている。だからこそ、目先の短期的な合理性ではなく、「長期的な合理性」を泥臭く取りに行く。それこそが、面白い社会を創り上げるための、最高に重要なファクターなんだ！
